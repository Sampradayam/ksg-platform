# SlotManager Backend

This is the **Slot Booking & Management** backend server built using **Node.js, Express, and MongoDB (MERN stack backend)**.

## Features

- **Slot Booking**
  - Create new bookings
  - View all bookings
- **Slot Management**
  - Add new slots
  - Get all slots
- **Authentication**
  - Middleware for protected routes (if implemented)
